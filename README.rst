TCPServer_For_Softimage - Addon
===============================

Installation
------------

Extract and Install like an usual Softimage Addon: Softimage -> File - > Addon -> Install Addon.

Usage
-----

TCPServer_For_Softimage menu entry is stored in View -> TCPServer -> TCPServer Preferences.
